A front-end developer is someone who creates websites and web applications. They use code that affects how a website looks and how a user interacts with its web pages. That includes simple elements such as buttons that take a visitor to different places within the website, as well as input methods, like a contact form1. Front-end developers are primarily responsible for the site’s appearance. Back-end developers work behind the scenes on the server-side of a website. They use various programming languages to make things happen, whether it’s going to a thank-you webpage for signing up for a newsletter or adding contact information to a database.
